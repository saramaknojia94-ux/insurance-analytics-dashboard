# Medicare Claims Analytics Dashboard

Started this project wanting to work with real, messy claims data 
rather than a clean toy dataset. CMS Medicare data is exactly that — 
millions of real provider claims with inconsistent formatting, 
outliers, and complexity that reflects actual healthcare analytics work.

## The Question
Where is Medicare spending the most money, which specialties drive 
the highest costs, and what does geographic variation look like 
across states?

## Data
CMS Medicare Physician & Other Practitioners dataset (2023)
Source: data.cms.gov
Real provider-level claims data — 9M+ rows

## Approach
- Pull data via CMS public API
- Clean and explore spending patterns
- Identify top cost drivers by specialty and state
- Build Tableau dashboard with KPI tiles and interactive filters

## Tools
Python, pandas, requests, Matplotlib, Tableau
